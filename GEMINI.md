# Project Overview

This is a simple graphics application written in Rust. It uses the `minifb` crate to create a window and render basic 2D shapes. The core logic is organized into a small library with modules for color representation and a framebuffer.

# Building and Running

*   **Build:** `cargo build`
*   **Run:** `cargo run`
*   **Test:** `cargo test`

# Development Conventions

The project follows standard Rust conventions. Code is formatted with `rustfmt` and checked with `clippy`. All contributions should adhere to these standards.
